Release Notes for USB Type-C™Power Delivery on STM32F0 software expansion for STM32Cube (UM2191)

Copyright 2018 STMicroelectronics

 


USB Type-C™ is the newest USB connector ecosystem, it addresses the evolving needs of platform and devices, while retaining the functional benefits of  USB.
USB Type-C™ Power Delivery on STM32F0 software expansion for STM32Cube consists of libraries (STM32F0 USB Type-C PD middleware stack, in binary format) and application examples for STM32F0 devices as USB Type-C port controllers, and will be extended to other STM32 Series in the future.

An example is also provided based on TCPM/TCPC architecture using an OnSemi FUSB307 TCPC controller.
     

The  X-CUBE-USB-PD software expansion for STM32 structure is layered on top of the HAL abd BSP drivers allowing a modular and flexible usage of the different modules. It can easily be tailored to different hardware platforms.
